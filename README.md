## Description

MetricMTG is a website for tracking and analyzing Commander games with detailed statistics, gameplay patterns, and comprehensive information on commanders and cards.

## Table of Contents

- [Features](#features)
- [Usage](#usage)
- [License](#license)


## Features

- Track Commander (EDH) games
  
- Detailed gameplay statistics and analysis
  
- Win rates based on various factors
  
- Database of commanders and cards
  
- Deck archetypes and strategies overview
  
- Personal Binder for decks and cards
  
- Opponents list with profiles and stats
  
- Featured deck carousel
  
- Explore and like community decks
  
- Recent games log with reports
  
- Leaderboard of top players
  
- Dark theme toggle


  
## Usage

In order to use the note taking application, you must:

- Sing in or register to the website

- Click Log new game to log a new game
